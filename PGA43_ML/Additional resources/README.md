# Additional Resources

This directory is a continuously curated, student-focused collection of end-to-end Data Science and Machine Learning learning modules.

## Learning-module standard

Each algorithm module should include:

1. What the algorithm is.
2. Why it exists and the problem it solves.
3. Mathematical intuition and derivation.
4. Assumptions and prerequisites.
5. How the algorithm works step by step.
6. From-scratch implementation using Python and NumPy where appropriate.
7. Library implementation using established packages such as scikit-learn.
8. Data preparation and exploratory analysis.
9. Model training, validation, and diagnostics.
10. Hyperparameter interpretation and tuning.
11. Evaluation metrics and error analysis.
12. When to use the algorithm.
13. When not to use it.
14. Failure modes, edge cases, and common mistakes.
15. Comparison with related algorithms.
16. A fully executed Jupyter notebook with rendered outputs.
17. Exercises, interview questions, and further-learning references.

## Directory convention

```text
Additional resources/
└── <algorithm-slug>/
    ├── README.md
    ├── theory/
    ├── notebooks/
    │   └── <algorithm-slug>-masterclass.ipynb
    ├── src/
    ├── data/
    ├── exercises/
    └── references.md
```

Every module must be self-contained, reproducible, technically accurate, and suitable for students learning the algorithm from first principles.
