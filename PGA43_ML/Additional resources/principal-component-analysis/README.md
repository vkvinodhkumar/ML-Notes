# Principal Component Analysis Masterclass

A self-contained student module for PCA from geometric, algebraic, statistical, and practical perspectives.

## Outcomes
Students will explain why PCA exists; derive it from maximum variance and minimum reconstruction error; connect covariance eigendecomposition with SVD; implement PCA from scratch; use scikit-learn in leakage-safe pipelines; choose component count; interpret scores, loadings, explained variance, reconstruction error, and whitening; diagnose scaling, outliers, instability, and information loss; and compare PCA with related methods.

## Learning path
1. Foundations
2. Mathematics and derivation
3. Data preparation and mechanics
4. Diagnostics, evaluation, and tuning
5. Comparisons, failure modes, and production
6. Executed notebook
7. Exercises and interview questions

The scratch implementation uses SVD. The library workflow covers scaling, component selection, reconstruction, and downstream classification.
