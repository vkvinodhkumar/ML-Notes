# Support Vector Machine Masterclass

A complete student-ready module for maximum-margin classification, soft margins, support vectors, kernels, diagnostics, tuning, and production use.

## Outcomes
Students will derive hard- and soft-margin SVM, explain hinge loss, slack variables, the dual and kernel trick, implement a binary linear SVM from scratch, build leakage-safe scikit-learn pipelines, tune linear and RBF models, and diagnose margins, support-vector count, calibration, class errors, and deployment cost.

## Learning path
1. Foundations
2. Mathematics and derivation
3. Data preparation and mechanics
4. Diagnostics, evaluation, and tuning
5. Comparisons, failure modes, and production
6. Executed notebook
7. Exercises and interview questions

## Structure
`theory/`, `src/`, `notebooks/`, `data/`, `exercises/`, `references.md`, `environment.yaml`, and `validate_module.py`.
