# References and further study

## Foundational literature

- Cover, T. M., and Hart, P. E. (1967). *Nearest Neighbor Pattern Classification*. IEEE Transactions on Information Theory.
- Fix, E., and Hodges, J. L. (1951). *Discriminatory Analysis: Nonparametric Discrimination: Consistency Properties*.

## Textbooks

- Hastie, T., Tibshirani, R., and Friedman, J. *The Elements of Statistical Learning*.
- James, G., Witten, D., Hastie, T., Tibshirani, R., and Taylor, J. *An Introduction to Statistical Learning*.
- Bishop, C. M. *Pattern Recognition and Machine Learning*.

## Implementation documentation

- scikit-learn User Guide: Nearest Neighbors.
- scikit-learn API documentation for `KNeighborsClassifier`, `KNeighborsRegressor`, `Pipeline`, `StandardScaler`, and `GridSearchCV`.

## Suggested progression

After mastering exact KNN, study nearest-centroid classifiers, metric learning, locality-sensitive hashing, approximate-nearest-neighbor indexing, vector databases, and retrieval evaluation.
