# AdaBoost Masterclass

A complete student-ready module for Adaptive Boosting.

## Outcomes
Students will understand weak learners, sample reweighting, weighted error, learner weight, exponential loss, margins, assumptions, diagnostics, tuning, failure modes, comparisons, and production use. The module includes a clean AdaBoost.M1 implementation from scratch and a scikit-learn implementation.

## Learning path
1. Foundations
2. Mathematics and derivation
3. Training mechanics and data preparation
4. Diagnostics, evaluation, and tuning
5. Comparisons, failure modes, and production
6. Executed notebook
7. Exercises and interview questions
