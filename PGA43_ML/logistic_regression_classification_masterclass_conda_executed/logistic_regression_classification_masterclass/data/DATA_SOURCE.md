# Dataset provenance

## Bundled file

`bank_marketing_teaching.csv` is generated by `scripts/generate_teaching_data.py` with random seed `43`.

It is a synthetic, schema-compatible teaching dataset inspired by the variable structure of the UCI Bank Marketing dataset. It is designed to provide:

- numerical and categorical predictors;
- an imbalanced binary target;
- interpretable log-odds relationships;
- repeated rows for a duplicate-audit exercise;
- a post-contact `duration` variable that demonstrates target leakage;
- enough observations for stratified splitting, cross-validation, inference, and diagnostics.

The file is not represented as original UCI data.

## Official dataset

The official Bank Marketing dataset is available from the UCI Machine Learning Repository. Run:

```bash
python scripts/download_official_data.py
```

The script downloads UCI dataset ID 222 and extracts `bank-full.csv` into this directory.

Citation:

Moro, S., Cortez, P., and Rita, P. (2014). *A Data-Driven Approach to Predict the Success of Bank Telemarketing*. Decision Support Systems, 62, 22–31.
