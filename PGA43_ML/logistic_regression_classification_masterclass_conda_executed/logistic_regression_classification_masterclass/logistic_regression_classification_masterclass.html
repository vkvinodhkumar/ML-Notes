Logistic Regression Classification Masterclass

This static companion mirrors the ready-to-open notebook package. Use the executed notebook for teaching and the clean notebook for student laboratories.

Notebook files

  • Executed instructor notebook
  • Clean student notebook

Coverage

Log-odds and sigmoid foundations; leakage-aware EDA; pipeline-safe preprocessing; baseline and regularized logistic regression; ROC and precision-recall analysis; calibration; class weighting and SMOTE; cost-sensitive thresholds; Statsmodels odds-ratio inference; residual and influence diagnostics; untouched-test evaluation; serialization and model-card guidance.

Environment

conda env create -f environment.yaml
conda activate logistic-regression-masterclass
python -m ipykernel install --user --name logistic-regression-masterclass --display-name "Python (Logistic Regression Masterclass)"

Validation

python scripts/verify_package.py
python scripts/reexecute_notebook.py

The dataset is generated deterministically on first notebook execution when data/bank_marketing_teaching.csv is absent.
